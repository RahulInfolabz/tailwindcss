Tailwind CSS Grid System

What are Grid System In Tailwind CSS?

The Grid System in Tailwind CSS is a utility-based approach to building responsive, structured layouts using CSS grid. Tailwind provides a set of utility classes to define grid containers, control rows and columns, and customize layout properties like alignment, spacing, and responsiveness.

Grid Classes in Tailwind CSS

Class Description
grid Defines a grid container.
grid-cols-{n} Sets the number of columns.
col-span-{n} Defines how many columns an Box spans.
gap-{n} Adds space between rows and columns.
grid-rows-{n} Sets the number of rows.
justify-* Aligns Boxs horizontally.
Boxs-* Aligns Boxs vertically.

1. Grid

To create a grid container, you use the grid class

Syntax : grid

Code :


<div class="grid bg-blue-300 p-5 mx-auto">
    <h1> Grid Container </h1>
</div>

Output:

Grid Container

2. Grid Column Classes

To create a grid layout, you can use the grid column classes

Syntax : grid-cols-{n}

here n number range can be (1-12)

Single Column Layout

Code :


<div class="container mx-auto grid grid-cols-1 ">
    <div class="border-2 border-black min-h-[200px] my-5">
        <h1>Box 1</h1>
    </div>
</div>

Output:

Box 1
